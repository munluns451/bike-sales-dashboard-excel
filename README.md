# 📊 Bike Sales Dashboard (Excel Project)

This project was created as part of my learning using a tutorial from Alex The Analyst (YouTube).
I followed the steps and practiced building dashboards using Excel.

I plan to build my own original projects next using real-world datasets.

## 📌 Overview
This project analyzes customer data to understand factors influencing bike purchases.

## 🎯 Objective
To identify patterns in income, age, and commute distance that affect buying decisions.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization

## 📊 Key Insights
- Higher income customers are more likely to purchase bikes
- Middle-aged customers show the highest purchase rate
- Short commute distances correlate with higher purchases
